What is Compactness

There's a bunch of various lemmas and definitions of compactness essentially imagine some set A defined on metric space X the set A is compact if there's no sequence in A which escapes it i.e. there's no holes or any other funny business going on with this set. For finite dimensional euclidean space this is just a closed and bounded set but more generally it's defined for any metric space as a generalization of this idea. Imagine some ball S that has no holes and a solid border any sequence in S has no way of escaping where as T has all these holes if a sequence enters one of these holes it can get out of T thus it's not compact.
